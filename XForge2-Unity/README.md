## 简介
XForge2在框架层面上完全抹除了两个引擎之间的差异。如果你是一个Unity初学者，你甚至不需要了解Unity中关于程序集和AssetBundle的相关知识，完全使用XForge2提供的工具和API即可完成与CocosCreator中一样的开发体验。
> Unity中的AssetBundle与CocosCreator中的AssetBundle有很多的差异，具体可以通过询问AI来了解。
